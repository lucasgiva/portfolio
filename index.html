Habilidades

html5

HTML (Hypertext Markup Language) é uma linguagem de marcação criada para web, que portanto é utilizada para estruturar a base de sites, ou seja, inserir elementos em sites através de tags, para que possa ser feita a navegação de usuários. Atualmente reuno bastante conhecimento sobre cada tag e o impacto que cada uma possui, assim uso-as de forma semântica e inteligente.

css3

CSS (Cascading Style Sheet), é um mecanismo e uma linguagem voltada para estilização dos componentes que estão inseridos em um site pelo HTML, o CSS é capaz de dar vida aos componentes modificando cores, tamanhos, posicionamento, animações, transições etc. Costumo utilizar o CSS de forma inteligente, busco o melhor estilo e combinação para ter um aspecto agradavel e para navegação ser satisfatória.

javascript

Javascript é por sua vez, uma linguagem de programação. na qual é utilizada para dar eventos e interações do usuário com o site. Me dedico para ser um bom programador javascript, tenho conhecimento sobre: tipos de dados, variáveis, estruturas de decisão, estruturas de repetição, arrays, objetos, funções. Sou capaz de manipular o DOM (Document Object Model) para fazer o usúario ter boas experiências ao entrar e interagir com o site.
